## French-Only Resources

Most graduate resources are French-first:
- graduate catalog hub
- admissions pages
- most faculty graduate pages
- many program PDFs

English pages exist for the university shell, but graduate detail content is usually French.

## Recommendations For Next Phase

1. Use the graduate catalog hub as the primary source list for program inventory.
2. Use the faculty hubs to resolve program ownership and later build the graduate program inventory.
3. Check the faculty-specific pages before assuming any graduate field is missing.
4. Search for tuition and scholarship pages only after the program list is stabilized.
5. Treat out-of-scope certificates/diplomas as separate records for future handling, not for the current graduate scope.

## Overall Recommendation

`APPROVE WITH NOTES`

USJ clearly offers graduate study and the main source surfaces have been identified. The discovery pass is sufficient to start the inventory phase, but tuition and several shared-support categories still need targeted follow-up.
