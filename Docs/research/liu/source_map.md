## Notable limitations

- The official site has multiple parallel/legacy URL structures for the same content (`LIU`, `NewLIU2022`, `SoASchool`, school-specific sub-sites, and metadata PDFs).
- Several plan-of-study links are embedded as images or icon links, so direct page crawling can miss PDFs unless discovered through search/indexed metadata.
- The Academic Calendar page was accessible, but extracted text did not include detailed calendar rows in this browser pass.
- Admission Leaflets PDF was accessible, but text extraction returned little/no structured text; snippets indicated master document requirements.
- A complete production import should perform a second pass over all official `NewLIU2022/metaData` PDFs to avoid missing any graduate program variants.
