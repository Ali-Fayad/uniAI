# Report Asset Index

| Asset family | Location | Purpose |
|---|---|---|
| Investigation | `investigation/` | Evidence, status, documentation gaps, traceability |
| Technical chapters | `technical/` | Report-ready technical source material |
| Diagrams | `diagrams/` | PlantUML source grouped by viewpoint |
| Tables | `tables/` | Independently reusable report tables |
| Screenshots | `screenshots/` | Captured images or manual-capture plan |
| APIs | `api/` | Controller-derived endpoint inventory |
| Testing | `testing/` | Automated and manual test evidence |
| Appendices | `appendices/` | Glossary, references, inventories, catalogues |

The principal database ERDs are [core application](diagrams/database/core_application_erd.puml) and [AI/knowledge](diagrams/database/ai_knowledge_erd.puml). They are logical views of one configured PostgreSQL database.
