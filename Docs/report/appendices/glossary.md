# Glossary

| Term | Meaning in uniAI |
|---|---|
| Citation | source URL/evidence returned alongside graduate retrieval context |
| Context budget | application limit that trims/rejects oversized AI requests |
| Deterministic fallback | local interpreter used when AI interpretation is unavailable/untrusted |
| Graduate retrieval | typed, SQL-backed information lookup for graduate data |
| Logical ERD view | documentation partition of one physical database |
| Prompt-visible memory | memory representation safe to send to AI prompts |
