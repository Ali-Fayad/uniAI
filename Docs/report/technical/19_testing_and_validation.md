# Testing and Validation

Backend tests use JUnit/Spring Boot Test, Mockito, H2, and Testcontainers. Graduate retrieval has unit tests for interpretation, validation, follow-ups, citations, ranking, compression, memory, and SQL adapter behavior, plus Docker-dependent integration tests. The frontend package defines lint/build scripts but no confirmed frontend test files.

See [automated results](../testing/automated_test_results.md) and [coverage gaps](../testing/coverage_gaps.md).
