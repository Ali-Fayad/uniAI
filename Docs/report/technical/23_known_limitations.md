# Known Limitations

- Google OAuth callback/code exchange is not confirmed.
- Wildcard CORS is explicitly a development TODO.
- Docker/Testcontainers validation is blocked where Docker is unavailable.
- The map is backed by static frontend coordinates, separate from the graduate catalogue.
- No frontend automated test suite was found.
- Academic-structure detailed retrieval needs explicit SQL bounding for production-scale data.
- AI answers depend on external provider availability and configured credentials/models.
