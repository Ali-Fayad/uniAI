# Project Overview

uniAI is a web platform for university-oriented guidance. The implemented product combines authenticated chat, configurable AI response generation, graduate-program retrieval, a CV Builder, catalog search, a map, feedback, and role-protected administration. The client is React/TypeScript/Vite; the server is Spring Boot/Java 17; persistence is PostgreSQL/Flyway.

Evidence: root `docker-compose.yml`, `Client/package.json`, `Server/pom.xml`.
